# Nexus.toml — Manifest Spec

> The single source of truth for a Nexus game. Engine version, style, genres, platforms, deploy targets, agent config — one file, machine-readable, human-editable.

## Boundaries
- Owns: the manifest schema, defaults, validation rules.
- Does NOT own: per-system internal configs (those live next to their crates).
- Depends on: `→ docs/architecture/03-tech-stack.md` (version pinning), `→ docs/specs/styles/overview.md`, `→ docs/specs/genres/`, `→ docs/specs/agent/sdk.md`.

## File Location

Repository root: `mygame/Nexus.toml`. Exactly one per project. Parsed by `nexus` CLI on every command. Re-validated by `nexus lint`.

## Top-Level Sections

| Section | Required | Purpose |
|---|---|---|
| `[project]` | yes | name, version, authors, license |
| `[engine]` | yes | nexus-engine version + feature flags |
| `[style]` | yes | visual style lock |
| `[genres]` | yes | genre modules enabled |
| `[platforms]` | yes | build targets |
| `[features]` | no | optional engine subsystems (physics, audio variants) |
| `[assets]` | no | asset pipeline config |
| `[scripting]` | no | script runtime selection |
| `[networking]` | no | netcode model + transport |
| `[server]` | no | server crate config |
| `[web]` | no | web sub-monorepo config |
| `[mobile]` | no | mobile companion config |
| `[infra]` | no | IaC backend + cloud target |
| `[dlc]` | no | DLC pack registry settings |
| `[mods]` | no | mod loader + sandbox config |
| `[agents]` | no | AI agent SDK config |
| `[deploy]` | no | per-target deploy spec |
| `[telemetry]` | no | structured telemetry sinks |
| `[ci]` | no | CI/merge bot config |

## Fully-Annotated Example

```toml
# Nexus.toml — generated by `nexus new mygame --genre=moba --style=npr`
# Edit freely. Re-validated on every `nexus` command. `nexus lint` enforces schema.

[project]
name            = "mygame"                  # kebab-case; max 64 chars
version         = "0.1.0"                   # semver
authors         = ["You <you@example.com>"]
license         = "MIT"                     # SPDX ID; default MIT
description     = "A weekend MOBA."
repository      = "https://github.com/you/mygame"
edition         = "2026"                    # Nexus template revision

[engine]
version         = "1.0"                     # nexus-engine semver; "*" for latest
features        = [                          # enabled engine subsystems
  "renderer",
  "physics",
  "audio",
  "networking",
  "scripting",
  "editor",                                  # omit for shipping builds to shrink binary
]
unsafe_allowed  = false                      # enforces principle: zero unsafe without justification
determinism     = "strict"                   # strict | best-effort | off  → docs/specs/physics/determinism.md

[style]
primary         = "npr"                      # pbr | npr | pixel | 2d | mixed
lock            = true                       # `nexus lint` rejects mixed-style assets
palette         = "assets/palette.png"       # required for `pixel`
references      = ["assets/style-refs/*.png"] # for AI asset generation conditioning

[genres]
primary         = "moba"                     # → docs/specs/genres/moba.md
secondary       = []                         # additional genre modules layered in
# Each genre listed here loads its module + scaffolds its conventions on `nexus generate`.

[platforms]
targets         = ["linux", "windows", "macos", "web"]   # subset of: linux|windows|macos|android|ios|web|switch|ps5|xbox
default         = "linux"
web_backend     = "webgpu"                                # webgpu | webgl2 (fallback)

[features]
physics         = "rapier"                   # rapier (default) | jolt-bridge [DECISION NEEDED]
audio           = "spatial"                  # spatial | stereo
networking      = "rollback"                 # rollback | replication | hybrid | off → docs/specs/networking/overview.md
ray_tracing     = false                      # opt-in; requires platform support

[assets]
source_dir      = "assets"
build_dir       = "target/assets"
compression     = { textures = "bc7", meshes = "meshopt", audio = "ogg" }
streaming       = true                                    # → docs/specs/assets/streaming.md
lod_auto        = true                                    # auto-LOD generation
generators      = ["kenney", "polyhaven", "meshy", "flux-local"]
generator_keys  = ".env"                                  # secrets file for paid APIs

[scripting]
runtime         = "lua"                      # lua | rune | both → docs/specs/scripting/overview.md
hot_reload      = true
scripts_dir     = "scripts"

[networking]
model           = "rollback"                 # → docs/specs/networking/rollback.md
transport       = "quic"                     # quic | udp
max_players     = 10
tick_rate_hz    = 60
relay_required  = true                       # use Nexus relay for NAT traversal

[server]
enabled         = true
mode            = "authoritative"            # authoritative | listen | p2p
crate           = "server"                   # path to server crate
deploy_target   = "k8s"                      # k8s | systemd | lambda | bare
image_base      = "ghcr.io/nexus/server-base:1.0"

[web]
enabled         = true
package_manager = "pnpm"                     # pnpm | bun  [DECISION NEEDED]
workspace_dir   = "web"
play_app        = "web/apps/play"
marketing_app   = "web/apps/marketing"

[mobile]
enabled         = true
framework       = "react-native"             # react-native | flutter | native
companion_only  = true                       # if true, mobile is companion, not the game itself

[infra]
backend         = "terraform"                # terraform | pulumi
cloud           = "aws"                      # aws | gcp | azure | hetzner | fly
regions         = ["us-east-1", "eu-west-1", "ap-southeast-1"]

[dlc]
enabled         = true
signing_key     = "keys/dlc.pub"             # public key; private key never in repo
registry_url    = "https://dlc.mygame.com"

[mods]
enabled         = true
runtime         = "rune"                     # rune | lua
sandbox         = "strict"                   # strict | permissive | off → docs/specs/scripting/sandbox.md
capabilities    = ["read_world", "spawn_entities", "play_sound"]
workshop_url    = "https://mods.mygame.com"

[agents]
sdk_version     = "1.0"                                   # → docs/specs/agent/sdk.md
config_dir      = "ai-agents"
headless_speed  = 100                                     # max sim speed multiplier
telemetry_port  = 9090                                    # JSON-RPC telemetry stream
scenarios_dir   = "tests/scenarios"
replay_dir      = "ai-agents/replays"
allowlist       = ["gameplay-dev", "balance-tuner", "qa-runner", "perf-engineer"]

[deploy]
# Per-target deploy. `nexus deploy <target>` selects.
  [deploy.steam]
  app_id        = 0                                       # set after Steamworks app reservation
  branch        = "default"                               # default | beta | playtest
  depot_map     = { linux = 1, windows = 2, macos = 3 }

  [deploy.itch]
  user          = "you"
  game          = "mygame"
  channels      = ["linux", "windows-zip", "web"]

  [deploy.web]
  provider      = "cloudflare"                            # cloudflare | vercel | netlify | s3
  domain        = "play.mygame.com"
  asset_cdn     = "cdn.mygame.com"

  [deploy.server]
  provider      = "fly"                                   # fly | aws-ecs | gcp-cloudrun | k8s-self
  app_name      = "mygame-server"
  regions       = ["iad", "fra", "syd"]
  scale_min     = 2
  scale_max     = 32

  [deploy.mobile]
  ios_team_id   = "ABCD1234EF"
  android_pkg   = "com.mygame.companion"

[telemetry]
sinks           = ["stdout-json", "otlp"]                 # → docs/specs/agent/telemetry.md
otlp_endpoint   = "https://otel.mygame.com:4317"
sample_rate     = 1.0                                     # 1.0 = every frame; reduce in prod
privacy_mode    = "anonymized"                            # raw | anonymized | off

[ci]
merge_bot       = "nexus-merge"                           # → docs/guides/merge-system.md
required_checks = ["nexus test", "nexus lint", "nexus build --target=all"]
auto_merge      = true                                    # AI merge bot can land PRs autonomously
human_review    = "optional"                              # required | optional | off
```

## Field Reference — Detail

### `[project]`

| Field | Type | Default | Notes |
|---|---|---|---|
| `name` | string | (required) | kebab-case, `^[a-z][a-z0-9-]{0,63}$` |
| `version` | semver | `0.1.0` | |
| `authors` | array<string> | `[]` | `"Name <email>"` |
| `license` | SPDX id | `MIT` | dual-license: `MIT OR Apache-2.0` |
| `description` | string | `""` | one line, used in store metadata |
| `repository` | URL | none | populated by `nexus new --git` |
| `edition` | string | `2026` | template revision; bumped by `nexus migrate` |

### `[engine]`

| Field | Type | Default | Notes |
|---|---|---|---|
| `version` | semver \| `*` | `*` | pinned via Cargo.lock for reproducibility |
| `features` | array<string> | see ex. | subset; omit `editor` for ship builds |
| `unsafe_allowed` | bool | `false` | principle: zero unsafe without justification |
| `determinism` | enum | `strict` | `strict` required if `networking.model = "rollback"` |

### `[style]`

| Field | Type | Default | Notes |
|---|---|---|---|
| `primary` | enum | (required) | `pbr` \| `npr` \| `pixel` \| `2d` \| `mixed` |
| `lock` | bool | `true` | rejects assets violating style at import |
| `palette` | path | none | required for `pixel` |
| `references` | array<path> | `[]` | feeds AI asset generation conditioning |

### `[genres]`

| Field | Type | Default | Notes |
|---|---|---|---|
| `primary` | enum | (required) | one of the genre modules — `→ docs/specs/genres/` |
| `secondary` | array<enum> | `[]` | layer multiple genres (e.g. `["rpg"]` on `primary="fps"`) |

### `[platforms]`

| Field | Type | Default | Notes |
|---|---|---|---|
| `targets` | array<enum> | `["linux"]` | each adds a `nexus build --target=X` capability |
| `default` | enum | first of `targets` | what `nexus build` (no flag) produces |
| `web_backend` | enum | `webgpu` | `webgl2` fallback when WebGPU absent |

### `[networking]`

| Field | Type | Default | Notes |
|---|---|---|---|
| `model` | enum | `off` | `rollback` \| `replication` \| `hybrid` \| `off` |
| `transport` | enum | `quic` | `quic` \| `udp` |
| `max_players` | u32 | `2` | tied to genre defaults (MOBA→10, BR→100) |
| `tick_rate_hz` | u32 | `60` | physics tick; clamp 20-240 |
| `relay_required` | bool | `false` | NAT traversal via Nexus relay |

### `[agents]`

| Field | Type | Default | Notes |
|---|---|---|---|
| `sdk_version` | semver | `1.0` | matches `nexus-agent-sdk` |
| `config_dir` | path | `ai-agents` | `.claude/agents/` lives inside |
| `headless_speed` | u32 | `1` | `nexus run --headless --speed=N` upper bound |
| `telemetry_port` | u16 | `9090` | JSON-RPC stream |
| `scenarios_dir` | path | `tests/scenarios` | `→ docs/specs/agent/scenarios.md` |
| `replay_dir` | path | `ai-agents/replays` | `→ docs/specs/agent/replay.md` |
| `allowlist` | array<string> | `[]` | which agent personas can be invoked |

## Validation Rules

| Rule | Error code |
|---|---|
| `project.name` matches regex | `E_NXTOML_001` |
| `engine.version` is valid semver | `E_NXTOML_002` |
| `style.primary` ∈ enum | `E_NXTOML_003` |
| at least one of `genres.primary` set | `E_NXTOML_004` |
| `platforms.targets` non-empty | `E_NXTOML_005` |
| if `networking.model = "rollback"` then `engine.determinism = "strict"` | `E_NXTOML_006` |
| if `style.primary = "pixel"` then `style.palette` set | `E_NXTOML_007` |
| `deploy.*` provider known | `E_NXTOML_008` |
| `agents.allowlist` ⊆ files in `ai-agents/.claude/agents/` | `E_NXTOML_009` |

All errors emitted as structured JSON when `--json` flag passed → AI-first mandate.

## Defaults Resolution

```
default Nexus.toml on `nexus new` (minimal):

[project]
name = "mygame"
version = "0.1.0"
license = "MIT"

[engine]
version = "*"
features = ["renderer", "physics", "audio"]

[style]
primary = "pbr"

[genres]
primary = "platformer"           # template's safest default

[platforms]
targets = ["linux", "web"]
```

Every other section uses the defaults documented above. Add sections as needed.

## Migration

`nexus migrate` bumps `project.edition` and applies known transforms between template versions. Always emits a diff preview before applying.

## Cross-Agent Flags
- `[AGENT: 02]` `engine.features` list must align with crate feature flags
- `[AGENT: 04]` `style.primary` enum must match style modules shipped
- `[AGENT: 07]` `networking.model` enum must match implemented netcode models
- `[AGENT: 12]` `genres.*` enum must match shipped genre modules
- `[AGENT: 10]` `agents.*` schema mirrors agent SDK config
