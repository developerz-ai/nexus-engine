# Style — 2D

> Modern 2D pipeline: sprite batching, tilemaps, parallax layers, normal-mapped 2D lighting with occluder shadows — for everything from Hollow Knight–scale platformers to Stardew-grade isometric games.

## Boundaries

- Owns: 2D camera + Y-sort, sprite/tile batchers, atlas packing rules, parallax stack, 2D light + occluder + normal-map shading, 2D post-FX defaults.
- Does NOT own: sprite asset import / atlasing internals (→ `docs/specs/assets/import.md`), low-level draw API (→ `docs/specs/renderer/backend.md`), pixel-art treatment (→ `docs/specs/styles/pixel.md`; the 2D style can be wrapped by `pixel`).
- Depends on: renderer overview, ECS transform (`docs/specs/core/ecs.md`), style overview.

## Architecture

```
2D Render Graph

  ECS (transforms)            Batcher                      Lighting pass            Compose
  ┌───────────────┐           ┌────────────────────┐       ┌──────────────────┐     ┌──────────┐
  │ Sprite2D      │  ───►     │ sort: layer asc,   │ ───►  │ for each Light2D │ ──► │ Y-sort   │
  │ Tilemap chunk │           │      Y desc (Y-srt)│       │   accumulate via │     │ across   │
  │ Parallax2D    │           │ pack into instance │       │   normal map ·   │     │ layers   │
  │ Light2D       │           │ buffer per atlas + │       │   light dir      │     │ ─► post  │
  │ Occluder2D    │           │ per shader perm    │       │ for each Occluder│     │ (bloom,  │
  │ ParticleEmit  │           │ → 1 draw call per  │       │   shadow mask    │     │  vignette│
  └───────────────┘           │   (atlas, perm)    │       │   (shadow vol)   │     │  bloom)  │
                              └────────────────────┘       └──────────────────┘     └──────────┘

Parallax layers: each layer = one Parallax2D node with scroll_scale ∈ [0,1].
                 Camera moves → layer offset = camera * scroll_scale.
```

```rust
// components
pub struct Sprite2D {
    pub texture: TextureHandle,
    pub region: Rect,                // atlas sub-rect
    pub size: Vec2,                  // world units
    pub anchor: Vec2,                // 0..1 in sprite local
    pub tint: Color,
    pub flip: Flip,
    pub layer: i16,                  // explicit ordering
    pub normal: Option<TextureHandle>,
    pub material: Material2DHandle,
}

pub struct Tilemap {
    pub tileset: TilesetHandle,      // grid + atlas
    pub chunks: ChunkStore,          // streamed (see assets/streaming.md)
    pub cell_size: Vec2,
    pub layer: i16,
}

pub struct Parallax2D {
    pub scroll_scale: Vec2,          // 0..1, 1 = locked to world
    pub repeat: BVec2,
}

pub struct Light2D {
    pub kind: Light2DKind,           // Point | Cone | Directional | FreeForm(polygon)
    pub color: Color,
    pub intensity: f32,
    pub radius: f32,
    pub falloff: Falloff,            // Linear | Inverse | InverseSquare
    pub cast_shadows: bool,
}

pub struct Occluder2D { pub polygon: Vec<Vec2>, pub self_shadow: bool }
```

## Performance Contract

| Metric | Target | Hard limit |
|--------|--------|------------|
| 10k sprites, 1 atlas, 1 light | < 2 ms | 6 ms |
| 100k sprites, batched, 1 atlas | < 8 ms | 16 ms |
| Tilemap chunk draw (1k tiles) | 1 draw call | 4 |
| 64 Light2D + 256 occluders, soft shadow | < 4 ms | 12 ms |
| Atlas page upload | < 1 ms (4096²) | 5 ms |
| GPU memory per atlas page (RGBA8 4096²) | 64 MB | — |
| Batcher CPU cost per 1k sprites | < 50 µs | 200 µs |

## Error Contract

| Code | Meaning | Caller action |
|------|---------|---------------|
| `STYLE_2D_E001` | Sprite references atlas page beyond `atlas_max_pages` | Re-pack atlases or raise limit |
| `STYLE_2D_E002` | Tilemap cell_size ≠ tileset cell | Fix tileset or cell_size |
| `STYLE_2D_E003` | `lighting.mode="normal_map"` but sprite missing normal | Provide normal or switch sprite mode |
| `STYLE_2D_E004` | More than `max_lights` active in frame | Cull or raise limit |
| `STYLE_2D_W001` | Sprite size not aligned to texel under `pixel` wrapping | Snap or accept blur |

## Integration Points

- **ECS**: components above register in core ECS; `Sprite2D` system is parallel-safe (read-only transforms). → `docs/specs/core/ecs.md`.
- **Renderer**: 2D pipeline is one render graph variant; reuses the standard color target. → `docs/specs/renderer/overview.md`.
- **Assets**: atlas packer runs at import; supports max-rects with rotation; produces a JSON manifest. → `docs/specs/assets/import.md`.
- **Pixel style**: when wrapped by `pixel` preset, the 2D pipeline draws into the low-res target unchanged. → `docs/specs/styles/pixel.md`.
- **Editor**: tilemap painter, atlas viewer, light/occluder gizmos. → `docs/specs/editor/scene.md`.
- **Physics**: 2D colliders + occluder polygons may share source data. → `docs/specs/physics/collision.md`.

## Normal-Mapped 2D Lighting

```
For each fragment of a sprite with a normal map:
  N  = sample(normal_map).xyz * 2 - 1     // tangent space, sprite-local
  for each Light2D within radius:
    L  = normalize(light.pos - frag.pos)
    d  = distance(light.pos, frag.pos)
    att = falloff(d, light.radius)
    if light.cast_shadows && in_shadow_volume(frag.pos, light.pos, occluders):
        att *= 0.0  (hard) or soft via PCF over shadow_steps
    diffuse += light.color * max(N · L, 0) * att
  color = albedo * (ambient + diffuse)
```

Sprite "tangent space" convention: +X right, +Y up (sprite-local), +Z out of screen. Tools like Laigter export with this convention by default; importer flag `normal_convention = "openGL2D"` enforces.

## Parallax Stack

- Each `Parallax2D` node holds a child subtree drawn at `offset = -camera.pos * scroll_scale`.
- `scroll_scale = 0` → static background; `1` → locked to world; `>1` → foreground (closer than camera).
- `repeat = (true, false)` tiles horizontally for endless side-scrollers.
- Lighting applies post-parallax; lights belong to a layer.

## Test Requirements

- 10k sprite scene at 60 FPS on integrated GPU (`[BENCHMARK NEEDED]` baseline machine).
- Y-sort: two sprites at different Y produce correct draw order with single-atlas batch (one draw call).
- Tilemap streams chunks on scroll without frame spikes &gt; 2 ms.
- Normal-map light produces a moving highlight that tracks light position (golden image rotation test).
- Occluder polygon casts a shadow whose silhouette matches polygon edges within 1 px at 1080p.
- Parallax: camera moves N units; far layer moves N × scroll_scale units (assertion).
- Atlas overflow → `STYLE_2D_E001` with `suggested_fix` pointing at offending sprite ids.

## Prior Art

- *Inspired by*: Godot 4 `CanvasItem` system + `Light2D` / `LightOccluder2D` (`docs.godotengine.org/en/stable/tutorials/2d/`). ✓ great editor integration; ✓ normal-map per `CanvasTexture`; ✗ batcher historically not optimal — Nexus targets instanced draws per (atlas, shader).
- *Inspired by*: Godot `Parallax2D` node (replacing old `ParallaxBackground`).
- *Inspired by*: Laigter (`azagaya.itch.io/laigter`) — open-source 2D normal-map generator; recommended companion tool, not bundled.
- *Inspired by*: LÖVE2D — simplest sprite batcher API ever shipped; ✓ minimal cognitive load. → `docs/prior-art/love2d.md`.
- *Inspired by*: Aseprite atlas export — informs the importer JSON schema.
