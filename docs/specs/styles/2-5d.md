# Style — 2.5D

> Three submodes for games that live between 2D and 3D. **HD-2D** (3D-rendered camera + 2D sprite layers — Octopath Traveler). **Parallax-2D** (multi-plane scrolling — Cuphead, Vanillaware). **Billboard-3D** (sprites face camera in 3D — Doom, Hotline Miami, Don't Starve). Each submode is a declarative profile under `[style] preset = "2-5d"` with a `submode` selector.

## Boundaries

- Owns: 2.5D submode registry, sprite-in-3D rules (depth-sort, alpha-test vs blend, soft particles vs sprites), per-submode camera contract, per-submode lighting model, shadow story (blob vs cast), per-submode performance budget.
- Does NOT own: low-level sprite batcher (→ `docs/specs/styles/2d.md`), low-res framebuffer / palette (→ `docs/specs/styles/pixel.md`), PBR pipeline (→ `docs/specs/styles/pbr.md`), per-layer style routing (→ `docs/specs/styles/mixed.md`).
- Depends on: `docs/specs/styles/overview.md`, `docs/specs/styles/2d.md`, `docs/specs/styles/mixed.md`, `docs/specs/renderer/overview.md`, `docs/specs/renderer/shadows.md`.

## Architecture

```
2.5D submode dispatch (selected at [style.2-5d] submode)

  ┌────────────────┐    ┌────────────────┐    ┌────────────────┐
  │   HD-2D        │    │ Parallax-2D    │    │ Billboard-3D   │
  │ (Octopath)     │    │ (Cuphead)      │    │ (Doom, HM)     │
  └───────┬────────┘    └───────┬────────┘    └───────┬────────┘
          │                     │                     │
   3D world, ortho cam   N flat planes,        Full 3D world,
   sprite chars on        each w/ scroll        sprites = quads
   billboards mixed       speed factor          locked to camera Y
   into G-buffer          composited 2D         (or full billboard)
          │                     │                     │
          ▼                     ▼                     ▼
   Renderer dispatches the correct render graph (one per submode).
   Mixed-style allowed: [[style.layer]] world=2-5d/hd-2d + ui=2d.
```

## Submode contracts

### HD-2D

| Concern | Contract |
|---|---|
| Camera | Orthographic OR locked-perspective (low FOV, ~25°). Pitch tilt 30°–60°. Roll = 0. |
| World geometry | Full 3D meshes (terrain, buildings) shaded PBR. |
| Characters | 2D sprites on camera-facing quads, depth-sorted into G-buffer. |
| Sprite depth | Quads write depth from sprite alpha-tested at α > 0.5. Soft particles between sprites use depth-aware compositor. |
| Lighting | 2D normal-map sprites sampled by 3D world lights. Spritelights cast on world via clustered-deferred. → `docs/specs/styles/pbr.md` shadow-mask path reused. |
| Shadows | Sprite-cast shadows: planar projection onto world. World-cast: standard PSSM. |
| Post | Tilt-shift bokeh recommended (defines the HD-2D look). Bloom on emissive sprite layers. |

### Parallax-2D

| Concern | Contract |
|---|---|
| Camera | Orthographic. Movement on X/Y only. Z = layer index. |
| World geometry | None. Layers are flat (sprites or single hand-painted plates). |
| Layer count | 2–16 planes. Each plane has `scroll_speed_x`, `scroll_speed_y` factors. |
| Lighting | Per-layer 2D lights (additive blend, normal maps optional per layer). |
| Shadows | Hand-painted into the plates. Engine does not generate. |
| Post | Per-layer or global. Vignette + grain typical. |

### Billboard-3D

| Concern | Contract |
|---|---|
| Camera | Full 3D, free movement, any FOV. |
| World geometry | Full 3D (BSP-style or mesh). |
| Sprites | Quads always face the camera. Two orientations: **full billboard** (rotates on all axes — Doom enemies) or **Y-billboard** (rotates only on world-up — top-down isometric, Don't Starve). |
| Sprite depth | Center-pivot quad writes depth from alpha-tested mask. Optional spherical-billboard for hi-fi shadows. |
| Lighting | Deferred lighting on sprites: sprite carries normal map (or autogen from silhouette). Shaded as if it were a 3D billboard imposter. |
| Shadows | Sprite casts axis-aligned bounding-box shadow (cheap) OR analytic capsule (closer match). Receives shadow maps as normal. |
| Post | Pixelation post-pass recommended for Doom-style retro look (chain with `docs/specs/styles/pixel.md`). |

## Public API

```toml
[style]
preset = "2-5d"

[style.2-5d]
submode = "hd-2d"           # "hd-2d" | "parallax-2d" | "billboard-3d"

# --- HD-2D ---
[style.2-5d.hd_2d]
camera        = "orthographic"    # "orthographic" | "locked-perspective"
camera_pitch  = 45.0              # degrees, only for locked-perspective
sprite_normal_maps = true         # use sprite normal maps under 3D lights
tilt_shift    = { strength = 0.6, focus_y = 0.4 }
sprite_shadows = "planar"         # "off" | "planar" | "blob"

# --- Parallax-2D ---
[style.2-5d.parallax]
layer_count   = 8
[[style.2-5d.parallax.layer]]
name = "sky"
scroll_x = 0.05
scroll_y = 0.0
[[style.2-5d.parallax.layer]]
name = "mountains"
scroll_x = 0.15
[[style.2-5d.parallax.layer]]
name = "ground"
scroll_x = 1.0

# --- Billboard-3D ---
[style.2-5d.billboard]
mode          = "full"            # "full" | "y-only"
alpha_test    = 0.5
sprite_lighting = "deferred"      # uses sprite normal maps
shadows       = "capsule"         # "off" | "aabb" | "capsule"
pixelation    = { enabled = true, internal_res = [640, 360] }
```

## Performance Contract

| Metric | HD-2D | Parallax-2D | Billboard-3D |
|---|---|---|---|
| Target frame budget @ 1080p | 16.6 ms (60 Hz) | 8.3 ms (120 Hz) | 16.6 ms (60 Hz) |
| Draw-call ceiling | 1500 | 200 (per layer batched) | 2000 |
| Sprite count active | 5000 | 5000 (per layer) | 10000 |
| Light count active | 64 dynamic | 32 per-layer | 128 dynamic |
| Memory: render targets | 1× backbuffer + tilt-shift bokeh half-res | N × layer color (8 × 8 MB @ 1080p RGBA8) | 1× backbuffer + GBuffer |
| Shadow pass cost | < 1.5 ms | n/a | < 2.0 ms |

## Error Contract

| Code | Meaning | Caller action |
|---|---|---|
| `STYLE_25D_E001` | Submode not specified or unknown | Set `[style.2-5d] submode = "hd-2d" | "parallax-2d" | "billboard-3d"` |
| `STYLE_25D_E002` | HD-2D requested with perspective camera + FOV > 40° | Lower FOV or switch to `submode = "billboard-3d"` |
| `STYLE_25D_E003` | Parallax layer count > 16 | Consolidate layers |
| `STYLE_25D_E004` | Billboard sprite missing normal map but `sprite_lighting = "deferred"` | Generate via `nexus assets billboard-normals <sprite>` or disable lighting |
| `STYLE_25D_E005` | Sprite shadow `"capsule"` requested but no capsule fitter present | Add capsule metadata to sprite asset or use `"aabb"` |
| `STYLE_25D_W001` | Sprite resolution mismatch — sprite is 32px but world is 4K → pixel cluster | Switch to HD-2D + tilt-shift OR upscale sprite |

## Integration Points

- **Renderer**: each submode binds a different RenderGraphId. HD-2D and billboard-3D use the deferred 3D graph; parallax-2D uses the 2D-batcher graph. → `docs/specs/renderer/overview.md`.
- **Styles/mixed**: `submode = "hd-2d"` plays well with `[[style.layer]]` — e.g. HD-2D world + 2D UI. → `docs/specs/styles/mixed.md`.
- **Styles/pixel**: billboard-3D + pixelation chain produces Doom-1993 look. → `docs/specs/styles/pixel.md`.
- **Particles**: heavy-particles SoftParticle mode handles sprite-particle depth bridging. → `docs/specs/renderer/particles-heavy.md`.
- **Physics**: sprite-in-3D collision is 2D capsule against 3D world. Standard `nexus-physics/character`. → `docs/specs/physics/character.md`.
- **Editor**: per-submode preview viewport. Live submode swap. → `docs/specs/editor/overview.md`.
- **Agent**: scenarios produce per-submode golden screenshots. → `docs/specs/agent/scenarios.md`.

## Test Requirements

- HD-2D: 3D terrain + 2D character sprite under a moving point light → normal-map lighting visible on sprite, planar shadow follows sprite position.
- Parallax-2D: 8-layer scene scrolls at correct ratios; camera moves 100 px → background moves 5 px.
- Billboard-3D: enemy sprite rotates to face camera at all yaw angles; alpha-test edges stable; standing behind world geometry → fully occluded.
- Pixelation chain on billboard-3D produces 640×360 internal → integer-upscaled to 1920×1080 with no filtering.
- Submode swap at runtime (`nexus run --hot-reload-style`) < 250 ms.
- Mixed-style + 2-5d: HD-2D world layer + 2D UI layer composite correctly with no depth bleed.
